# Required Packages

- cjson
- curl
- unzip

## Installing cjson

### Arch linux
```bash
sudo pacman -S cjson
```

### Debian
```bash
sudo apt install libcjson-dev
```

## Fedora
```bash
sudo dnf install cjson
```

### MacOS(Unsupported)

#### Homebrew(https://brew.sh)
```bash
brew install cjson
```
