# Required Packages

- cjson
- curl
- unzip
- pkg-config (build)

## Installing cjson

### Arch linux
```bash
sudo pacman -S cjson
```

### Debian
```bash
sudo apt install libcjson-dev
```

## Fedora
```bash
sudo dnf install cjson
```

#### Homebrew(Unsupported)
```bash
brew install cjson
```

## Installing curl

### Arch linux
```bash
sudo pacman -S curl
```

### Debian
```bash
sudo apt install libcurl4-openssl-dev curl
```
